Add custom buttons to the HTML editor toolbar and full tabbing functionality.

== Description ==

This plugins allows you to add your own custom buttons to the HTML editor toolbar. For example, to add an H1, H2, H3 button to make adding headers to your content easier. Add as many buttons as you need.

And it adds an 'enable / disable tabs' button for tabbing single and multi-line text directly within the HTML editor. You can easily indent using the tab key or unindent using the (shift+tab) key combo.

The plugin also includes a clean uninstall on deactivation. For support, feature requests, or to report a bug, visit our contact form at <a href="http://uniquemethod.com/contact">Unique Method</a>.

== Installation ==

1. Upload the 'html-editor-reloaded' directory to the '/wp-content/plugins/' directory
2. Activate the plugin on the 'Plugins' page
3. Go to the 'Settings' panel and click on 'HTML Editor' from the menu
